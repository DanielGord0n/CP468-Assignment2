# Person E - Results and Testing Analyst

## Responsibilities
- Test the program using various Sudoku puzzles of different difficulty levels
- Record and analyze:
  - Number of steps until consistency
  - Queue lengths per iteration
  - Whether the puzzle was solved by AC-3 alone or needed the secondary solver
- Prepare tables and charts summarizing the findings for inclusion in the report
- Provide interpretations of the results and discuss computational performance

## Deliverables
- `results/` folder with all test outputs and logs
- Report section: Results and Performance Analysis
